📊 Audit Dashboard Project

🔍 Overview
This project demonstrates a realistic financial audit using a dummy dataset and an interactive dashboard.

📁 Files Included
- audit_dataset.csv → Dataset used for analysis
- Audit Dashboard.html → Interactive dashboard
- Audit Dashboard Analysis.pdf → Detailed audit report

📊 Dashboard Features
- KPI metrics (Total Transactions, Risk Count)
- Fraud and anomaly detection
- Department-wise insights
- Risk categorization (High, Medium, Low)

🧠 Key Insights
- High-risk transactions identified
- Duplicate entries detected
- Unapproved transactions flagged
- Late payments observed

Tools Used
- Data Analytics
- Dashboard Visualization
- CSV Dataset
